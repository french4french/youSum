Introduction

L'API YouSummarize vous permet de générer automatiquement des résumés de vidéos YouTube en utilisant l'intelligence artificielle. Notre API offre un moyen programmatique d'accéder aux fonctionnalités principales de notre service.

L'API peut être utilisée pour récupérer des informations sur une vidéo YouTube, extraire sa transcription et générer un résumé structuré de son contenu.

URL de base de l'API

https://api.yoursitedomain.com

En local, l'API est accessible à http://localhost:3000

Authentification

Actuellement, l'API YouSummarize ne nécessite pas d'authentification pour un usage local ou en développement. Pour un déploiement en production, il est recommandé d'implémenter une authentification API par clé.

Note importante

Bien que l'API ne nécessite pas d'authentification, elle utilise les API YouTube Data et Google Gemini, qui requièrent des clés API valides. Ces clés doivent être configurées dans le fichier .env de votre serveur.

POST

/api/summarize

Génère un résumé structuré d'une vidéo YouTube en utilisant l'IA de Google Gemini.

Corps de la requête (JSON)

Paramètre Type Requis Description
transcription string Oui Transcription de la vidéo à résumer
videoInfo object Non Informations sur la vidéo (titre, chaîne, etc.)
language string Non Langue du résumé (fr, en). Par défaut: fr
isShorter boolean Non Indique si la transcription est partielle. Par défaut: false

Gestion des erreurs

L'API utilise les codes d'état HTTP standard pour indiquer le succès ou l'échec d'une requête. En cas d'erreur, la réponse inclut un message expliquant la cause du problème.

Code Description Exemple
200 Succès La requête a réussi
400 Mauvaise requête {"error": "ID de vidéo requis"}
404 Non trouvé {"error": "Vidéo non trouvée"}
500 Erreur serveur {"error": "Erreur serveur", "message": "..."}

Limites d'utilisation

L'API YouSummarize n'impose pas de limites d'utilisation strictes de son côté. Cependant, notre service est basé sur les API de Google (YouTube Data API et Gemini API) qui ont leurs propres limitations :

API Limite Notes
YouTube Data API 10 000 unités/jour Chaque requête consomme des unités (quota). Consultez la documentation YouTube.
Google Gemini API Varie selon le forfait Les limites dépendent de votre compte Google AI. Consultez la documentation Gemini.

Conseil d'optimisation

Pour optimiser votre utilisation, nous recommandons de mettre en cache les résultats des requêtes fréquentes, particulièrement pour les vidéos populaires. Cela réduira la consommation de quota des API tierces.

Exemples d'intégration

Exemple Node.js complet

Voici un exemple d'utilisation de l'API avec Node.js pour résumer une vidéo YouTube : 

const axios = require('axios');

// Configuration
const API_BASE_URL = 'http://localhost:3000'; // Remplacer par votre URL d'API en production
const YOUTUBE_VIDEO_ID = 'dQw4w9WgXcQ';

// Fonction principale asynchrone
async function summarizeYouTubeVideo(videoId, language = 'fr') {
  try {
    // 1. Récupérer les informations de la vidéo
    console.log('Récupération des informations de la vidéo...');
    const videoInfoResponse = await axios.get(`${API_BASE_URL}/api/video-info`, {
      params: { videoId }
    });

    const videoData = videoInfoResponse.data;
    const videoInfo = {
      title: videoData.items[0].snippet.title,
      channelTitle: videoData.items[0].snippet.channelTitle,
      publishedAt: videoData.items[0].snippet.publishedAt,
      duration: convertDuration(videoData.items[0].contentDetails.duration)
    };

    console.log(`Vidéo trouvée: "${videoInfo.title}" par ${videoInfo.channelTitle}`);

    // 2. Récupérer la transcription
    console.log('Récupération de la transcription...');
    const transcriptResponse = await axios.get(`${API_BASE_URL}/api/transcript`, {
      params: { videoId, language }
    });

    const transcription = transcriptResponse.data.transcript;
    console.log(`Transcription récupérée: ${transcription.length} caractères`);

    // 3. Générer le résumé
    console.log('Génération du résumé...');
    const summarizeResponse = await axios.post(`${API_BASE_URL}/api/summarize`, {
      transcription,
      videoInfo,
      language
    });

    // 4. Afficher le résumé
    const summary = summarizeResponse.data.candidates[0].content.parts[0].text;
    console.log('\n=== RÉSUMÉ ===\n');
    console.log(summary);

    return summary;
  } catch (error) {
    console.error('Erreur lors du résumé de la vidéo:', error.response?.data?.error || error.message);
    throw error;
  }
}

// Fonction utilitaire pour convertir la durée ISO 8601 en format lisible
function convertDuration(isoDuration) {
  const match = isoDuration.match(/PT(\d+H)?(\d+M)?(\d+S)?/);

  const hours = (match[1]) ? parseInt(match[1].slice(0, -1)) : 0;
  const minutes = (match[2]) ? parseInt(match[2].slice(0, -1)) : 0;
  const seconds = (match[3]) ? parseInt(match[3].slice(0, -1)) : 0;

  let result = '';
  if (hours > 0) result += `${hours}h `;
  if (minutes > 0 || hours > 0) result += `${minutes}m `;
  result += `${seconds}s`;

  return result.trim();
}

// Exécuter la fonction
summarizeYouTubeVideo(YOUTUBE_VIDEO_ID)
  .catch(err => console.error('Échec de l\'opération:', err));

Exemple d'intégration frontend

Voici un exemple d'intégration de l'API dans une application web frontend avec JavaScript vanilla : 

// Fonction pour résumer une vidéo YouTube
async function summarizeVideo(videoUrl, language = 'fr') {
  // Afficher l'état de chargement
  updateStatus('Extraction de l\'ID de la vidéo...');

  // Extraire l'ID de la vidéo depuis l'URL
  const videoId = extractVideoId(videoUrl);
  if (!videoId) {
    throw new Error('URL YouTube invalide');
  }

  // 1. Récupérer les informations de la vidéo
  updateStatus('Récupération des informations de la vidéo...');
  const videoInfoResponse = await fetch(`/api/video-info?videoId=${videoId}`);

  if (!videoInfoResponse.ok) {
    const errorData = await videoInfoResponse.json();
    throw new Error(errorData.error || 'Erreur lors de la récupération des informations');
  }

  const videoData = await videoInfoResponse.json();
  const videoInfo = {
    title: videoData.items[0].snippet.title,
    channelTitle: videoData.items[0].snippet.channelTitle,
    publishedAt: videoData.items[0].snippet.publishedAt,
    duration: videoData.items[0].contentDetails.duration
  };

  // 2. Récupérer la transcription
  updateStatus('Récupération de la transcription...');
  const transcriptResponse = await fetch(`/api/transcript?videoId=${videoId}&language=${language}`);

  if (!transcriptResponse.ok) {
    const errorData = await transcriptResponse.json();
    throw new Error(errorData.error || 'Erreur lors de la récupération de la transcription');
  }

  const transcriptData = await transcriptResponse.json();
  const transcription = transcriptData.transcript;

  // 3. Générer le résumé
  updateStatus('Génération du résumé avec l\'IA...');
  const summarizeResponse = await fetch('/api/summarize', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      transcription,
      videoInfo,
      language
    })
  });

  if (!summarizeResponse.ok) {
    const errorData = await summarizeResponse.json();
    throw new Error(errorData.error || 'Erreur lors de la génération du résumé');
  }

  const summaryData = await summarizeResponse.json();
  const summary = summaryData.candidates[0].content.parts[0].text;

  // Mettre à jour l'UI
  updateStatus('Résumé généré avec succès!', 'success');
  displaySummary(summary, videoInfo, videoId);

  return summary;
}

// Fonction pour extraire l'ID de la vidéo depuis une URL YouTube
function extractVideoId(url) {
  if (!url) return null;

  // Gestion des URLs de visionnage
  const watchRegex = /(?:youtube\.com\/watch\?v=|youtu\.be\/)([a-zA-Z0-9_-]{11})/;
  const watchMatch = url.match(watchRegex);
  if (watchMatch) return watchMatch[1];

  // Gestion des URLs d'intégration
  const embedRegex = /youtube\.com\/embed\/([a-zA-Z0-9_-]{11})/;
  const embedMatch = url.match(embedRegex);
  if (embedMatch) return embedMatch[1];

  return null;
}

// Fonctions pour mettre à jour l'interface utilisateur
function updateStatus(message, type = 'info') {
  const statusElement = document.getElementById('status');
  statusElement.textContent = message;
  statusElement.className = `status ${type}`;
}

function displaySummary(summary, videoInfo, videoId) {
  const resultsElement = document.getElementById('results');

  // Créer un élément pour le résumé avec mise en forme Markdown
  const summaryElement = document.createElement('div');
  summaryElement.className = 'summary-content';
  summaryElement.innerHTML = markdownToHtml(summary);

  // Afficher les informations de la vidéo
  const videoInfoElement = document.createElement('div');
  videoInfoElement.className = 'video-info';
  videoInfoElement.innerHTML = `
    
    
${videoInfo.title}

    
Chaîne: ${videoInfo.channelTitle}

    
Publié le: ${new Date(videoInfo.publishedAt).toLocaleDateString()}

    Voir sur YouTube
  `;

  // Vider et remplir l'élément de résultat
  resultsElement.innerHTML = '';
  resultsElement.appendChild(videoInfoElement);
  resultsElement.appendChild(summaryElement);
  resultsElement.classList.remove('hidden');
}

// Fonction pour convertir le Markdown en HTML (version simplifiée)
function markdownToHtml(markdown) {
  return markdown
    .replace(/^# (.*$)/gim, '
$1
')
    .replace(/^## (.*$)/gim, '
$1
')
    .replace(/^### (.*$)/gim, '
$1
')
    .replace(/\*\*(.*?)\*\*/g, '$1')
    .replace(/\*(.*?)\*/g, '$1')
    .replace(/\n/g, '
');
}

// Gestionnaire d'événement pour le formulaire
document.getElementById('summarizeForm').addEventListener('submit', async function(e) {
  e.preventDefault();

  const videoUrl = document.getElementById('videoUrl').value;
  const language = document.getElementById('language').value;

  try {
    // Désactiver le bouton pendant le traitement
    const submitButton = document.getElementById('submitButton');
    submitButton.disabled = true;

    // Résumer la vidéo
    await summarizeVideo(videoUrl, language);
  } catch (error) {
    updateStatus(`Erreur: ${error.message}`, 'error');
    console.error('Erreur:', error);
  } finally {
    // Réactiver le bouton
    document.getElementById('submitButton').disabled = false;
  }
});

Auto-hébergement et personnalisation

YouSummarize est un projet open-source que vous pouvez déployer et personnaliser selon vos besoins. Pour héberger votre propre instance de l'API, suivez ces étapes :

1. Prérequis

  • Node.js (v14 ou supérieur)
  • npm (v6 ou supérieur)
  • Un compte Google Cloud avec les API YouTube Data et Gemini activées
  • Des clés API valides pour ces services

2. Installation

# Cloner le repository
git clone https://github.com/votre-utilisateur/youSummarize.git
cd youSummarize

# Installer les dépendances
npm install

# Configurer les variables d'environnement
cp .env.example .env
# Modifiez le fichier .env avec vos propres clés API

3. Lancement du serveur

npm start

Votre API sera accessible à l'adresse: http://localhost:3000

4. Personnalisation

Vous pouvez personnaliser le comportement de l'API en modifiant les fichiers suivants :

  • services/transcript.js - Pour modifier la logique d'extraction des transcriptions
  • server.js - Pour ajouter ou modifier des endpoints API
  • .env - Pour configurer les clés API et autres paramètres
